Лабораторная работа №5 по Java, вариант 4.

Вариант 4 по таблице:
1.1, 2.1, 3.1, 4.1, 5.1, 6.5, 7.1, 7.2

Структура выбора в Main:
- сначала вводится номер задания
- затем номер задачи
- используется конструкция if-else

Запуск:
1) Откройте папку проекта.
2) Компиляция:
   javac -encoding UTF-8 -d out src/ru/student/lab5/Main.java src/ru/student/lab5/*/*.java
3) Запуск:
   java -cp out ru.student.lab5.Main

Для заданий с файлами можно ввести 0, чтобы использовать sample-файлы из папки data.
